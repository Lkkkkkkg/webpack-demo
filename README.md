# webpack-demo
记录自己学习webpack时练手的demo

## 项目运行
```
克隆仓库
git clone https://github.com/Lkkkkkkg/webpack-demo.git
进入相对应demo文件夹
cd demo01
安装依赖
npm install
启动
npm run build
```

demo01 - 初步安装webpack, 简单配置了webpack

demo02 - 对加载css进行了配置

demo03 - 对加载图片进行了配置

demo04 - 对加载字体进行了配置

demo04 - 对加载数据文件(JSON, XML, CSV等)进行了配置
